Path Candles

A logo designed for a candle company.

While we're not a design firm by any stretch (there are much more talented artists out there than us) on occasion if really pushed we'll make a logo. This is one we did for a startup attempting to launch a particular type of candle. While we're not sure what happened to their business, we know their logo is timeless.

  • Project: Path Candles Logo
  • Client:Path Candles Ltd.
  • Category:Design
  • Date:15/11/2014

Designed in Adobe Illustrator to allow for image scaling, the clients could then use the file however they needed it without worrying about the image size.

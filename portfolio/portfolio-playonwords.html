PlayOnWords.co

A website for a copywriting agency.

PlayOnWords is actually a project put together for NoBull's in-house copywriter, Chanelle, as a homepage for her own Copywriting agency. Looking for something that was 'clear and concise while offering [her] services in an enticing way' we're pretty happy with the results.

  • Project: PlayOnWords.co
  • Client:Chanelle MS
  • Category:Website
  • Date:15/09/2016

Built in HTML5/CSS/JS using the Bootstrap framework, the site was designed to be easily editable while still loading quikly and requiring little maintenance.
